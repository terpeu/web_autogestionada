web autogestionada menu del día
===============================
Listado de requisitos
---------------------
Desde una web o un móvil poder consultar los menús del día de un restaurante
Lo actualizará una persona con conocimientos básicos de informática
1.Mostrar los diferentes  menús del día (mínimo 3)
2.Mostrar sólo el día presente
3.El menú está estructurado : primeros,  segundos, postres, bebidas, incluidos y precio 
4.Mostrar datos genéricos del restaurante: Nombre, dirección, teléfono, email, whasapp y web, horario de cocina,logotipo y fotos genéricos
